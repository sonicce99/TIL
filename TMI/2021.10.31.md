# Vue 사용하기

### CDN 으로 사용하기  

```html
  <script src="https://unpkg.com/vue@next"></script>

  <div id="app">
    <h1>{{ message }}</h1>
  </div>
```


```javascript
  Vue.createApp({
    data: function () {
      return {
        message: 'Hello Vue!'
      }
    }
  }).mount('#app')
```


### NPM으로 시작하기

```
vue create (원하는 폴더명)
```
