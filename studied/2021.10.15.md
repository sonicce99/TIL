# 내가 알아 둬야 하는 함수

***

### match

```javascript
  const a = '1S'

  const b = a.match(/\dS/);
  const c = a.match(/(\d)S/);
  const d = a.match(/\d(S)/);
  const e = a.match(/(\d)(S)/);


  console.log(b,c,d,e)

  // expected output :
(1) ["1S"]
(2) ["1S", "1"]
(2) ["1S", "S"]
(3) ["1S", "1", "S"]
```

알아둬야 할 점 : 기본 졍규식과 똑같은데 괄호가 있을 경우 해당 조건을 만족하는 전체 문자열과 괄호안을 만족하는 문자열이 같이 반환됨.
